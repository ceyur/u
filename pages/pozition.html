свойство: значение; – объяснение свойства, его особенности Значения: 1. 2. пример (значение по умолчанию) Подобные свойства: 1. 2.

Обозначения: 1. ↓ – свойство наследуется 2. ■ – объединение

Значения, приминяющиеся для всех свойств: 1. inherit – элемент должен унаследовать родительское значение 2. initial – элемент должен использовать значение по умолчанию (не браузерное) 3. revert – элемент должен использовать браузерное значение, а не по умолчанию 4. revert-layer – откатывает значение свойства к значению свойства в предыдущем каскадном слое 5. unset – для наследуемых свойств значение работает так же, как inherit; для ненаследуемых свойств значение работает так же, как initial

Позиционирование

position: значение; – задаёт способ позиционирования элемента относительно окна браузера или других объектов Значения: 1. static – элемент позиционируется относительно родителя и соседних элементов 2. relative – элемент позиционируется как static, но его можно двигать относительно этого положения 3. absolute – элемент позиционируется относительно ближайшего родителя с relative, absolute, fixed или sticky 4. fixed – элемент позиционируется относительно окна браузера 5. sticky – элемент позиционируется как static но когда верхняя граница элемента будет находиться на расстоянии, указанном в параметре top, от верхней границы окна браузера (либо указанном в параметре bottom от нижней), он останется в этом положении относительно окна до тех пор пока его нижняя граница не упрется в другой sticky элемент или конец родительского элемента position: absolute; (static)

top: размер; – задаёт размер верхнего внутреннего отступа для элемента со значениями позиционирования relative, absolute, fixed и sticky top: 50px; (auto)

right: размер; – задаёт размер правого внутреннего отступа для элемента со значениями позиционирования relative, absolute, fixed и sticky right: 50px; (auto)

bottom: размер; – задаёт размер нижнего внутреннего отступа для элемента со значениями позиционирования relative, absolute, fixed и sticky bottom: 50px; (auto)

left: размер; – задаёт размер левого внутреннего отступа для элемента со значениями позиционирования relative, absolute, fixed и sticky left: 50px; (auto)

z-index: значение; – задаёт наложение элементов по оси Z (0 – элемент снизу, 1 – элемент сверху) для элемента со значениями позиционирования relative, absolute и fixed, можно указывать любое занчение z-index: 100; (auto)
