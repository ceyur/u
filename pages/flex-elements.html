свойство: значение; – объяснение свойства, его особенности Значения: 1. 2. пример (значение по умолчанию) Подобные свойства: 1. 2.

Обозначения: 1. ↓ – свойство наследуется 2. ■ – объединение

Значения, приминяющиеся для всех свойств: 1. inherit – элемент должен унаследовать родительское значение 2. initial – элемент должен использовать значение по умолчанию (не браузерное) 3. revert – элемент должен использовать браузерное значение, а не по умолчанию 4. revert-layer – откатывает значение свойства к значению свойства в предыдущем каскадном слое 5. unset – для наследуемых свойств значение работает так же, как inherit; для ненаследуемых свойств значение работает так же, как initial

Флекс-элементы

order: номер■flex-элемента; – управляет порядком, в котором располагаются flex-элементы внутри flex-контейнера order: 1; (0)

align-self: значение; – переопределяет для конкретного flex-элемента выравнивание вдоль поперечной оси Значения: 1. auto – flex-элементы растягиваются на весь размер поперечной оси flex-контейнера 2. flex-start – flex-элементы прижимаются к началу поперечной оси 3. flex-end – flex-элементы прижимаются к концу поперечной оси 4. center – flex-элементы располагаются по центру поперечной оси align-self: flex-end; (auto)

flex-basis: размер; – задаёт размер элемента по умолчанию, до распределения оставшегося пространства flex-basis: 300px; (auto)

flex-grow: значение; – управляет возможностью элемента увеличиваться в размере, заполняя весь flex-контейнер Значения: 1. 0 – запрещено увеличиваться в размере 2. 1 – разрешено увеличиваться в размере flex-grow: 1; (0)

flex-shrink: значение; – управляет возможностью элемента уменьшаться в размере, заполняя весь flex-контейнер Значения: 1. 0 – запрещено уменьшаться в размере 2. 1 – разрешено уменьшаться в размере flex-shrink: 0; (1)

flex: flex-grow flex-shrink flex-basis; – сокращённая запись для свойств flex-grow, flex-shrink и flex-basis flex: 1 0 auto; (0 1 auto)
