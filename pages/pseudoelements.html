свойство: значение; – объяснение свойства, его особенности Значения: 1. 2. пример (значение по умолчанию) Подобные свойства: 1. 2.

Обозначения: 1. ↓ – свойство наследуется 2. ■ – объединение

Значения, приминяющиеся для всех свойств: 1. inherit – элемент должен унаследовать родитесльское значение 2. initial – элемент должен использовать значение по умолчанию (не браузерное) 3. revert – элемент должен использовать браузерное значение, а не по умолчанию 4. revert-layer – откатывает значение свойства к значению свойства в предыдущем каскадном слое 5. unset – для наследуемых свойств значение работает так же, как inherit; для ненаследуемых свойств значение работает так же, как initial

Псевдоэлементы

элемент:before {контент} – применяется для вывода какого-либо контента перед элементом, к которому он добавляется Особенности: 1. При добавлении :before к блочному элементу, значение свойства display может быть только: block, inline, none, list-item, остальные значения будут трактоваться, как block. 2. При добавлении :before к строчному элементу, display ограничен значениями inline и none, остальные будут трактоваться, как inline. a:before { content: "🌵"; }

элемент:after {контент} – применяется для вывода какого-либо контента после элементом, к которому он добавляется Особенности: 1. При добавлении :after к блочному элементу, значение свойства display может быть только: block, inline, none, list-item, остальные значения будут трактоваться, как block. 2. При добавлении :after к строчному элементу, display ограничен значениями inline и none, остальные будут трактоваться, как inline. a:after { content: "🍣"; }

элемент::selection {свойства} – задаёт стиль выделенной части документа body::selection { background: green; }

элемент:first-line {свойства строки} – задаёт стиль первой строки текста, но к ней применяются не все стили p:first-line { font-size: 25px; }

элемент:first-letter {свойства символа} – задаёт стиль первого символа текста, но к нему применяются не все стили p:first-letter { font-weight: 900; }

элемент::-ms-clear {свойства} – задаёт стиль кнопки для очистки поля ввода в Internet Explorer, которая появляется в правой части поля только при вводе текста input::-ms-clear { display: none; }

элемент::-moz-focus-inner {свойства} – задаёт стиль внутренней части кнопки в FireFox button::-moz-focus-inner { padding: 0; border: 0; }
