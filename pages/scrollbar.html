свойство: значение; – объяснение свойства, его особенности Значения: 1. 2. пример (значение по умолчанию) Подобные свойства: 1. 2.

Обозначения: 1. ↓ – свойство наследуется 2. ■ – объединение

Значения, приминяющиеся для всех свойств: 1. inherit – элемент должен унаследовать родительское значение 2. initial – элемент должен использовать значение по умолчанию (не браузерное) 3. revert – элемент должен использовать браузерное значение, а не по умолчанию 4. revert-layer – откатывает значение свойства к значению свойства в предыдущем каскадном слое 5. unset – для наследуемых свойств значение работает так же, как inherit; для ненаследуемых свойств значение работает так же, как initial

Свойства скроллбара

scrollbar-width: значение; – задаёт ширину полосы прокрутки (скроллбара) Не работает в WebView на Android и в Safari Значения: 1. auto – ширина скроллбара по умолчанию 2. thin – боллее тонкая полоса прокрутки 3. none – полоса прокрутки исчезнет, однако элемент по-прежнему будет прокручиваться scrollbar-width: none; (auto)

↓scrollbar-color: значение■или■цвет значение■или■цвет; – задаёт цвет ползунка и дорожки Не работает в WebView на Android и в Safari Значения: 1. auto – цвет скроллбара по умолчанию scrollbar-color: lime green; (auto)

элемент::-webkit-scrollbar {свойства} – свойства применяются ко всей полосе прокрутки (скроллбару) Не работает в FireFox ::-webkit-scrollbar { background: blue; }

элемент::-webkit-scrollbar-track {свойства} – свойства применяются только к полосе прокрутки (дорожке) Не работает в Safari на мобильных устройствах и в FireFox .object::-webkit-scrollbar-track { width: 20px; }

элемент::-webkit-scrollbar-track-piece {свойства} – свойства применяются только к видимой части дорожки, не занятой ползунком Не работает в Safari на мобильных устройствах и в FireFox p::-webkit-scrollbar-track-piece { opacity: 0.5; }

элемент::-webkit-scrollbar-thumb {свойства} – свойства применяются к перетаскиваемому ползунку прокрутки Не работает в Safari на мобильных устройствах и в FireFox button::-webkit-scrollbar-thumb { border: 3px solid #fff; }

элемент::-webkit-scrollbar-button {свойства} – свойства применяются к кнопкам сверху и снизу дорожки, прокручивающие по одной строке Не работает в Safari на мобильных устройствах и в FireFox *::-webkit-scrollbar-button { height: 17px; }

элемент::-webkit-scrollbar-corner {свойства} – свойства применяются к уголку на скроллбаре, где пересекаются вертикальная и горизонтальная дорожки Работает только с теми элементами, чьи размеры пользователь может изменять (cвойство resizer), не работает в Safari на мобильных устройствах и в FireFox #view::-webkit-scrollbar-corner { background: pink; }

элемент::-webkit-resizer {свойства} – свойства применяются кнопке изменения размеров элемента в уголке на скроллбаре, где пересекаются вертикальная и горизонтальная дорожки Работает только с теми элементами, чьи размеры пользователь может изменять (cвойство resizer), не работает в FireFox div::-webkit-resizer { border-bottom: 10px solid #000; }

элемент:horizontal {свойства} – псевдокласс определяющий, что свойства применятся только к горизонтальному элементу Работает только с элементами скроллбара webkit div::-webkit-scrollbar-track:horizontal { box-shadow: inset 0 1px 10px green; }

элемент:vertical {свойства} – псевдокласс определяющий, что свойства применятся только к горизонтальному элементу Работает только с элементами скроллбара webkit div::-webkit-scrollbar-thumb:vertical { background: #118; }

элемент:start {свойства} – псевдокласс определяющий, что свойства применятся к элементу только до ползунка прокрутки (идентично decrement) Работает только с элементами скроллбара webkit div::-webkit-scrollbar-thumb:start { background: red; }

элемент:end {свойства} – псевдокласс определяющий, что свойства применятся к элементу только после ползунка прокрутки (идентично increment) Работает только с элементами скроллбара в webkit div::-webkit-scrollbar-thumb:end { box-shadow: 5px 5px 5px grey; }

элемент:window-inactive {свойства} – псевдокласс определяющий, что свойства применятся только тогда, когда окно не активно (находится сзади) Работает только с элементами скроллбара в webkit div::-webkit-scrollbar-thumb:window-inactive { overflow: clip; }
